# package.json 依存関係説明

## 概要
COOOLa クラウド型倉庫管理システム - フロントエンド設定
AngularJS 1.8.x + Bootstrap 5.x を使用したWebアプリケーション

## npmスクリプト定義

| スクリプト | 説明 |
|-----------|------|
| `start` | 開発サーバー起動（ポート3000、キャッシュ無効） |
| `build` | ビルドスクリプト（現在はダミー） |
| `test` | テストスクリプト（現在はダミー） |
| `lint` | リントスクリプト（現在はダミー） |

## プロジェクトキーワード（npm検索用）

- `warehouse` - 倉庫管理
- `management` - 管理システム
- `angularjs` - AngularJSフレームワーク
- `inventory` - 在庫管理

## 開発依存関係（開発時のみ必要）

| パッケージ | バージョン | 説明 |
|-----------|-----------|------|

## テーブル機能について

**注意**: `ng-table`パッケージは存在しないため削除しました。

**代替案**:
- Bootstrap 5のネイティブテーブル機能を使用
- `angular-ui-bootstrap`のテーブルコンポーネントを使用
- カスタムテーブルディレクティブを作成

## インストール方法

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm start
```

## 注意事項

- package.jsonは有効なJSON形式である必要があります
- JavaScriptコメント（`//`）は使用できません
- 依存関係のバージョンは`^`記号で指定し、互換性のある最新版を使用します
- 存在しないパッケージは事前に確認してから追加してください
